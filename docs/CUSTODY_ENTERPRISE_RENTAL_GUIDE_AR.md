# دليل التأجير — بوابة تأجير المعدات (منفصلة عن التأجير القديم)

> خلف علم `client_rental_portal_enabled` (**معطّل** حتى اعتماد النص القانوني والتسعير).
> جداول جديدة `custody_rental_*` — لا تلمس نظام التأجير القديم.

## التدفق
طلب تأجير → مراجعة التوفّر → عرض/موافقة سعر → عقد → **توقيع العميل** → استلام بصور →
عهدة تأجير نشطة → طلب إرجاع → فحص → رسوم تلف/تأخير (مرجعية) → إغلاق.

## المطلوب
- عميل/مستأجر (فرد/جهة) + تواصل. مدة التأجير. عربون/تأمين **مرجعي** (لا مبالغ فعلية بلا اعتماد مالي).
- قائمة المعدات + صور قبل/بعد + توقيع العميل والموظف + عقد PDF ورقم عقد + شروط.
- **لا تعود المعدة للمخزون قبل الاعتماد.**

## الخصوصية والصلاحيات
- RLS: المستأجر يرى **عقده ومستنداته فقط**. المستندات الحساسة في private storage.
- لا تخزين بيانات هوية أكثر من اللازم (مرجع فقط).

## الحالة
schema + RPC (إنشاء طلب) جاهز؛ العقد/التوقيع/الفحص وواجهة البوابة تُستكمل بعد اعتماد القانوني/التسعير.
