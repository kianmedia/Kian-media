# ملاحظات الإصدار — منصّة إدارة المشاريع (Project Core) · الطور 3 → 8D

منصّة إدارة مشاريع متكاملة (Next.js 14 + Supabase/PostgREST، عربيّة RTL) تغطّي
المشروع الواحد، المشاريع الكبيرة الهرمية، البرامج المستمرّة، والمشاريع الصغيرة —
بمصدر حقيقة واحد لكل بُعد (المرحلة، التقدّم، التخطيط، الموارد، الحوكمة، الإغلاق).

## الميزات حسب الطور

- **الطور 3 — المهام والتقدّم**: محرّك مهام (إسناد/اعتماديات/مهام فرعية)، Kanban
  وسير عمل، تقدّم موحّد + صحّة + تنبيهات، دلالات اعتماديات وجاهزية مرحلة وتقارير.
- **الطور 4 — التخطيط والموارد**: محرّك Gantt V2 (بلا Temp Tables)، خطوط أساسية،
  سعة موارد وحجوزات ومحرّك تعارض، تسوية ومحفظة وإغلاق التخطيط.
- **الطور 5 — الحوكمة والتنفيذيّ والإغلاق**: المخاطر/المشكلات/القرارات/طلبات
  التغيير + اعتمادات بـSLA (5A)؛ لوحات المحفظة التنفيذية والمؤشّرات (5B)؛
  الإغلاق/القبول/الدروس/إعادة الفتح/الأرشفة المؤسسية (5C).
- **الطور 6 — الهرمية**: تفعيل master ⇄ subproject (6A)، تجربة الهرمية والتنقّل
  والتجميع (6B)، دمج الإغلاق في المسارات التنفيذية (6C).
- **الطور 7 — القوالب والعمليات**: نظام قوالب وإصدارات وحفظ-كقالب خادميّ (7A)،
  مركز عمليات يوميّ يُركّب المصادر القائمة (7B).
- **الطور 8 — القابلية للتوسّع**:
  - **8A** إعدادات البرنامج وبيانات الوحدات + لوحة برنامج مشتقّة.
  - **8B** مخطّط الموجة المتدرّجة والإنشاء الجماعيّ للوحدات (idempotent).
  - **8C** المسار السريع للمشاريع الصغيرة (معالج بخطوتين، لوحة مشتقّة، إظهار تدريجيّ).
  - **8D** التزامات البرامج ومحرّك قياس SLA ومصفوفة تسليم العميل + ملخّص برنامج العميل.
- **التثبيت النهائيّ**: إصلاح تراكميّ إضافيّ واحد (`project_platform_stabilization_RUNME.sql`)
  يغلق تسريب عزل في `exec_gov_counts`، ويصحّح مفردات المشكلات/التغييرات في 5B،
  ويضيف قفلًا ضدّ تكرار الاعتماد المتزامن.
