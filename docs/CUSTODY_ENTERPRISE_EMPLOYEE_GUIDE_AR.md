# دليل الموظف — عهدتي المسجلة

بوابة كيان ← «عهدتي المسجلة». تبويبان: **صرف عهدة** و**إرجاع العهدة** (والتطبيق على الجوال أسرع للتصوير).

## صرف عهدة (فوري بلا انتظار موافقة)
1. ابحث عن الأصل أو امسح QR (على الجوال).
2. اختر الأصل/الكمية (المتاح فقط).
3. **صورة إلزامية لكل قطعة** + **صورة إجمالية**.
4. وافق على الإقرار ووقّع (إن كان مفعّلًا).
5. «تأكيد صرف العهدة» ← تصبح نشطة باسمك فورًا.

## إرجاع العهدة
1. افتح العهدة النشطة.
2. لكل قطعة: **صورة إرجاع** + الحالة (سليمة/بها ملاحظات/تالفة/ناقصة/مفقودة) + ملاحظة إن لم تكن سليمة.
3. **صورة إجمالية** للمرتجع.
4. «إرسال طلب الإرجاع» ← ينتظر فحص المسؤول (لا يُغلق حتى الاعتماد؛ إن رُفض تُصحّح وتعيد الإرسال).

## الإبلاغ عن مشكلة
داخل العهدة: نوع الحادث + وصف + صور + موقع (اختياري). يُوضع الأصل قيد المراجعة تلقائيًا.

## جلسة الموقع (إن كانت مفعّلة)
تبدأ بموافقتك لجلسة المهمة فقط، وتنتهي عند إغلاقها. لا تعمل طوال الوقت. ترى سجلّ جلساتك.

## لا تظهر لك
قيم الشراء/الفواتير/التأمين، بيانات موظفين آخرين، عهد غيرك.
